# python-finder
#### 功能介绍
>1、选择文件夹，展示该文件夹及子文件夹下的所有文件  
>2、选择要过滤的文件类型  
>3、选择要模糊匹配的内容  
>4、展示匹配到的内容，并将查询结果写入选择文件夹下的result.txt中
